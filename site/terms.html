About this tool

Cutter is a personal automation tool that downloads YouTube videos, cuts them into short clips, and posts them to TikTok and Instagram. It is operated by Christopher Lacy-Hulbert for personal use.

This tool is not a public service and is not offered to third parties. Access to its functionality is restricted to the operator.

Data collection and use

Cutter does not collect, store, or process personal data belonging to any third party. It operates exclusively on content owned or controlled by the operator.

When connecting to TikTok or Instagram, the tool uses OAuth tokens to act on behalf of the operator's own accounts. These tokens are stored locally on the operator's machine and are not transmitted to any third-party service beyond the platform APIs themselves.

Video content processed by this tool originates from YouTube and is handled locally. No content is shared with, sold to, or accessible by anyone other than the operator.

Platform API usage

Cutter interacts with the TikTok Content Posting API and the Meta Graph API in accordance with each platform's developer terms. It does not scrape, automate user interactions, or access data beyond what is required to post content to the operator's own accounts.

Use of this tool is subject to the terms of service of any third-party platform it connects to, including TikTok and Instagram.
